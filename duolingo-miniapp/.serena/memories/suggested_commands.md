# Suggested Commands

## Basic Repo Navigation (Linux)
- `ls`
- `cd <path>`
- `rg "pattern"` (ripgrep)
- `find <dir> -name "pattern"`
- `git status`

## Running the App
- Open the project in WeChat Developer Tools (GUI) and click `编译` / `预览`.

## Optional TDesign
- `npm install tdesign-miniprogram`
